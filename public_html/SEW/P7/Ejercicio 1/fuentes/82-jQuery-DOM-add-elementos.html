jQuery-DOM: Ejemplo de manipulación de elementos del árbol DOM - añadir elementos - eliminar elementos

append(): Inserta contenidos al final de los elementos seleccionados

prepend(): Inserta contenido en el inicio de los elementos seleccionados

after(): Inserta contenido después de los elementos seleccionados

before(): Inserta contenido antes de los elementos seleccionados

remove(): Elimina los elementos seleccionados (y sus descendientes)

Ejemplo desarrollado por Juan Manuel Cueva Lovelle. Departamento de Informática. Universidad de Oviedo
